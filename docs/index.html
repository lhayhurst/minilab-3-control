Ableton Live 12 · Remote Script

iamkat MiniLab 3

A custom control surface script that puts the right Live controls exactly where your hands already are.

↓ Download Install View on GitHub
"I wrote this for a musician I work with for whom the stock MiniLab 3 script didn't work: they wanted a toggle to arm the track, a toggle to move between input monitoring states, and a simpler way to move around tracks — turning the rotary encoder, rather than shift-turning it."

Installation

1
Set your MiniLab 3 to DAW mode
Hold Shift and press the 3rd Prog pad. The display will show DAW. The script requires DAW mode — it will not work in the default Arturia mode.
2
Download the latest release
Grab iamkat_MiniLab_3.zip from the Releases page.
3
Unzip and move the folder
Double-click the zip to unzip it, then move the iamkat_MiniLab_3 folder to:
~/Music/Ableton/User Library/Remote Scripts/

Tip: press ⌘ Shift G in Finder and paste the path above to jump straight there.
4
Select the script in Ableton
Restart Ableton Live, open Preferences → MIDI, and set Control Surface to iamkat_MiniLab_3. The Input port may auto-select to Minilab3 MIDI — if not, set it manually.
5
Enable the MIDI input
Still in Preferences → MIDI, find Minilab3 MIDI in the Input list and turn on both Track and Remote.
6
That's it
The buttons light up with color-coded LEDs when the controller connects.

2
Download the latest release
Grab iamkat_MiniLab_3.zip from the Releases page.
3
Unzip and move the folder
Right-click the zip → Extract All, then move the iamkat_MiniLab_3 folder to:
C:\Users\[your username]\Documents\Ableton\User Library\Remote Scripts\

Tip: paste that path directly into the Windows Explorer address bar.
4
Select the script in Ableton
Restart Ableton Live, open Options → Preferences → MIDI, and set Control Surface to iamkat_MiniLab_3. The Input port may auto-select to Minilab3 MIDI — if not, set it manually.
5
Enable the MIDI input
Still in Preferences → MIDI, find Minilab3 MIDI in the Input list and turn on both Track and Remote.
6
That's it
The buttons light up with color-coded LEDs when the controller connects.
Controller mode note: This script requires DAW mode. To switch modes, hold Shift and press the Prog button — each press cycles through the available modes. Keep pressing until the display shows DAW. If you ever want to return to the default Arturia mode, press Shift + Prog again to cycle back.
